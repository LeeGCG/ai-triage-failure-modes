# Failure Mode Analysis

## Observed Failure Types

### 1. Overconfidence
The model provides confident recommendations even when uncertain.

### 2. Ambiguity Failure
The model struggles when symptoms are unclear or conflicting.

### 3. Under-triage Risk
Some serious conditions are not prioritized appropriately.

### 4. Misinterpretation
The model takes user framing at face value without sufficient skepticism.

## Key Insight
AI systems may appear reliable in clear cases but degrade significantly in ambiguous or adversarial scenarios.
